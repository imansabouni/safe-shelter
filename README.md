# Safe Shelter

A comprehensive Disaster Shelter Management System developed as a Computer Engineering Graduation Project.

## Overview

Safe Shelter is a disaster management platform designed to improve coordination between shelters, disaster victims, volunteers, and administrators during emergency situations.

The project consists of:

- 🌐 Web Admin Panel
- 📱 Mobile Application
- ⚙️ Laravel REST API Backend

## Technologies

### Backend
- Laravel
- PHP
- MySQL
- Laravel Sanctum

### Web
- React
- TypeScript
- Vite
- Tailwind CSS

### Mobile
- React Native
- Expo

## Main Features

- Shelter Management
- QR Check-in / Check-out
- Family Card System
- Resource Management
- Donation Tracking
- Emergency Notifications
- Help Requests
- Offline Maps
- Smart Navigation
- Disaster Dashboard
